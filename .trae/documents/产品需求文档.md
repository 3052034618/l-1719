
# 汽车租赁集团智能调度与运营管理系统 - 产品需求文档

## 1. 产品概述

大型汽车租赁集团智能调度与运营管理桌面系统，实现车辆全生命周期管理，涵盖智能调度、预订管理、维保管理、费用结算、统计分析五大核心模块，通过数据驱动提升车队运营效率。

- 核心目标：实现车辆调度智能化、运营流程自动化、决策支持数据化
- 目标用户：集团调度经理、门店运营人员、财务人员、维保工程师
- 核心价值：提升车辆出租率20%、降低运营成本15%、提升客户满意度

## 3. 核心流程

### 3.1 智能调度流程

系统每日凌晨自动生成当日车辆分配方案，综合考虑车型匹配、客户信用、2小时清洁时间、车队负载均衡等因素，经经理审批后推送至各门店终端。

```mermaid
flowchart LR
    A["实时数据接入"] --> B["调度算法计算"]
    B --> C["生成初步方案"]
    C --> D["经理审批"]
    D -->|通过 --> E["推送门店终端"]
    D -->|驳回 --> F["重新调度"]
    F --> C
```

### 3.2 预订取车流程

客户预订后，取车时自动校验驾照、违章记录和信用分，低于阈值需预缴押金，校验通过后生成电子合同。

```mermaid
flowchart TD
    A["客户预订"] --> B["到达门店取车"]
    B --> C["驾照校验"]
    C -->|不通过 --> D["拒绝取车"]
    C -->|通过 --> E["违章查询"]
    E -->|有违章 --> F["预缴押金"]
    E -->|无违章 --> G["信用分评估"]
    F --> G
    G -->|低于阈值 --> H["补缴押金"]
    G -->|达标 --> I["生成电子合同"]
    H --> I
    I --> J["车辆交付"]
```

### 3.3 还车结算流程

还车时检查车辆状态，自动结算费用（基础租金+超时滞纳金+其他费用），生成发票，更新车辆状态。

```mermaid
flowchart TD
    A["客户还车"] --> B["车辆检查"]
    B --> C["费用结算"]
    C --> D["基础租金"]
    C --> E["超时累进滞纳金"]
    C --> F["其他费用"]
    D + E + F --> G["总费用计算"]
    G --> H["押金抵扣/退还"]
    H --> I["生成电子发票"]
    I --> J["车辆状态更新"]
    J --> K["进入清洁队列"]
```

### 3.4 维保管理流程

根据车辆里程自动触发生成维保工单，维保过程中扣减配件库存，完成后更新车辆状态。

```mermaid
flowchart TD
    A["里程数据采集"] --> B["维保阈值判断"]
    B -->|达到阈值 --> C["自动生成工单"]
    C --> D["分配维保工程师"]
    D --> E["配件出库"]
    E --> F["维保执行"]
    F --> G["工单完成确认"]
    G --> H["车辆状态更新"]
    H --> I["库存扣减记录"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝渐变（#1e3a5f 到 #2563eb），专业稳重
- **辅助色**：琥珀色（#f59e0b）用于强调、绿色（#10b981）用于成功状态、红色（#ef4444）用于警示
- **中性色**：深灰到浅灰的完整色阶，保证内容层次感
- **按钮风格**：圆角中等（rounded-lg），悬停时轻微阴影提升
- **字体**：Inter 字体族，清晰易读
- **布局风格**：左侧导航 + 顶部状态栏 + 主内容区，卡片式布局
- **图标风格**：Lucide 线性图标，简洁现代

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 驾驶舱首页 | 数据概览 | 渐变统计卡片、实时地图、数据表格、状态指示灯 |
| 智能调度中心 | 调度方案 | 时间线调度视图、方案对比、审批操作栏 |
| 预订管理 | 预订列表 | 筛选栏、数据表格、状态标签、详情弹窗 |
| 车辆管理 | 车辆列表 | 车辆卡片、状态筛选、地图视图切换 |
| 维保管理 | 工单管理 | 工单卡片、进度条、配件列表 |
| 取还车管理 | 流程向导 | 步骤指示器、表单、合同预览 |

- 桌面端优先设计，适配 1920px 最佳展示
- 支持平板端自适应，最小支持 1280px 宽度
- 侧边栏可折叠，提供更多内容空间

### 4.4 交互体验

- 数据加载使用骨架屏动画
- 状态变化使用平滑过渡动画
- 表格支持键盘快捷键操作
- 重要操作有确认提示
- 实时数据刷新有动态指示
